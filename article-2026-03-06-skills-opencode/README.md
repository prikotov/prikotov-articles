## О статье

Эксперимент с SKILLs в OpenCode + GLM-5: как я выстроил процесс создания статей для блога. Data-driven подход, автоматизация аналитики, честные выводы.

## Структура

```
research/
  2026-03-06 - SKILLs и процесс создания статей/
    01 - План исследования.md      # Этап 1: планирование
    02 - Инсайты.md                # Этап 3: анализ данных
    03 - Стратегия статьи.md       # Этап 4: стратегия
    04 - План статьи.md            # Этап 5: план

```
